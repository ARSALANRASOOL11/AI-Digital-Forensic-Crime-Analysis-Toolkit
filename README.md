# AI Digital Forensic Crime Analysis Toolkit

## Overview

The AI Digital Forensic Crime Analysis Toolkit is a Python-based forensic investigation platform designed to assist investigators in analyzing digital evidence using artificial intelligence and computer vision techniques.

The system can process uploaded evidence, classify potential crimes, perform face recognition, and generate investigation reports to support forensic analysis.

## Features

* AI-Based Crime Classification
* Digital Evidence Analysis
* Face Recognition System
* Image Processing and Object Detection
* Automated Investigation Reports
* Evidence Management Dashboard
* User-Friendly Web Interface
* Real-Time Evidence Upload and Analysis

## Installation

### Clone the Repository

```bash
git clone https://github.com/ARSALANRASOOL11/AI-Digital-Forensic-Crime-Analysis-Toolkit.git
```

### Navigate to the Project Directory

```bash
cd AI-Digital-Forensic-Crime-Analysis-Toolkit
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```

## Workflow

1. Upload digital evidence.
2. AI analyzes the uploaded content.
3. Crime-related objects and patterns are detected.
4. Evidence is classified into relevant crime categories.
5. Reports are generated for investigation purposes.
